# 북마크 일괄 삭제 기능 수동 테스트 체크리스트

> 서버 실행 후 브라우저에서 직접 확인할 수 있는 체크리스트입니다.

## 사전 준비

- [ ] 개발 서버 실행 (`pnpm dev`)
- [ ] Clerk 로그인 완료
- [ ] 북마크가 최소 3개 이상 있는 계정으로 로그인 (테스트용)

---

## 1. 기본 UI 확인

### 1.1 북마크 페이지 접근

- [ ] `/bookmarks` 페이지로 이동
- [ ] 페이지 제목 "내 북마크" 표시 확인
- [ ] 정렬 옵션 버튼 (최신순, 이름순, 지역별) 표시 확인

### 1.2 체크박스 UI 표시

- [ ] 목록 상단에 "전체 선택" 체크박스 표시 확인
- [ ] 각 북마크 카드 좌측 상단에 체크박스 표시 확인
- [ ] 체크박스가 카드 위에 올바르게 배치되어 있는지 확인

---

## 2. 체크박스 선택 기능

### 2.1 개별 선택

- [ ] 첫 번째 카드의 체크박스 클릭

  - [ ] 체크박스가 체크됨
  - [ ] 카드에 하이라이트 효과 (테두리, 그림자, 약간 확대) 표시 확인
  - [ ] 상단에 "1개 선택됨" 텍스트 표시 확인
  - [ ] 상단에 "삭제" 버튼 (빨간색, 휴지통 아이콘) 표시 확인

- [ ] 두 번째 카드의 체크박스 클릭

  - [ ] "2개 선택됨"으로 변경 확인
  - [ ] 두 카드 모두 하이라이트 표시 확인

- [ ] 세 번째 카드의 체크박스 클릭
  - [ ] "3개 선택됨"으로 변경 확인

### 2.2 개별 해제

- [ ] 체크된 카드의 체크박스 다시 클릭
  - [ ] 체크박스 해제됨
  - [ ] 카드 하이라이트 제거됨
  - [ ] 선택된 개수 감소 확인

### 2.3 전체 선택

- [ ] 모든 체크박스 해제 상태에서 상단 "전체 선택" 체크박스 클릭
  - [ ] 모든 카드의 체크박스가 체크됨
  - [ ] 모든 카드에 하이라이트 표시됨
  - [ ] "N개 선택됨" (N = 전체 북마크 수) 표시 확인

### 2.4 전체 해제

- [ ] 전체 선택 상태에서 상단 "전체 선택" 체크박스 다시 클릭
  - [ ] 모든 체크박스 해제됨
  - [ ] 모든 카드 하이라이트 제거됨
  - [ ] "삭제" 버튼 사라지고 "전체 선택" 헤더만 표시됨

---

## 3. 일괄 삭제 기능

### 3.1 삭제 버튼 표시/숨김

- [ ] 체크박스 선택 전: "삭제" 버튼이 보이지 않음
- [ ] 체크박스 1개 이상 선택: "삭제" 버튼 표시됨
- [ ] 모든 체크박스 해제: "삭제" 버튼 사라짐

### 3.2 확인 다이얼로그

- [ ] 2개 이상의 북마크 선택
- [ ] "삭제" 버튼 클릭
  - [ ] 다이얼로그 팝업 표시됨
  - [ ] 제목: "북마크 삭제 확인" 표시 확인
  - [ ] 내용: "선택한 N개의 북마크를 삭제하시겠습니까?" 표시 확인
  - [ ] 내용: "이 작업은 되돌릴 수 없습니다." 표시 확인
  - [ ] "취소" 버튼 표시 확인
  - [ ] "삭제" 버튼 (빨간색) 표시 확인

### 3.3 다이얼로그 취소

- [ ] 다이얼로그에서 "취소" 버튼 클릭
  - [ ] 다이얼로그 닫힘
  - [ ] 선택된 체크박스 유지됨
  - [ ] 북마크 목록 변경 없음

### 3.4 삭제 실행

- [ ] 2개 이상의 북마크 선택
- [ ] "삭제" 버튼 클릭
- [ ] 다이얼로그에서 "삭제" 버튼 클릭
  - [ ] "삭제" 버튼에 로딩 스피너 표시됨
  - [ ] "삭제 중..." 텍스트 표시됨
  - [ ] 버튼이 비활성화됨 (클릭 불가)

### 3.5 삭제 완료

- [ ] 삭제 실행 후
  - [ ] 다이얼로그 자동으로 닫힘
  - [ ] 우측 상단에 토스트 메시지 표시됨
    - [ ] 메시지: "N개의 북마크가 삭제되었습니다." (N = 삭제된 개수)
    - [ ] 초록색 성공 아이콘 표시
  - [ ] 페이지가 자동으로 새로고침됨
  - [ ] 삭제된 북마크가 목록에서 사라짐
  - [ ] 선택 상태 초기화됨 (체크박스 모두 해제)

## 빠른 확인 체크리스트 (5분 버전)

> 시간이 부족할 때 핵심만 빠르게 확인하는 체크리스트

### 필수 확인 사항

1. [ ] 북마크 페이지 접근 (`/bookmarks`)
2. [ ] 체크박스가 각 카드에 표시되는지 확인
3. [ ] 체크박스 클릭 시 카드 하이라이트 표시 확인
4. [ ] 상단에 "N개 선택됨" 및 "삭제" 버튼 표시 확인

6. [ ] 다이얼로그에서 "삭제" 클릭 시 삭제 실행 확인
7. [ ] 삭제 후 토스트 메시지 표시 및 목록 새로고침 확인
