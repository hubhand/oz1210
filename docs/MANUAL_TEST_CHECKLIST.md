# My Trip - 수동 테스트 체크리스트

> 개발 서버 실행 후 브라우저에서 직접 확인할 수 있는 기능 테스트 리스트입니다.
> 각 항목을 체크하면서 실제 동작을 확인하세요.

## 🚀 서버 실행 전 확인사항

- [ ] `.env` 파일이 존재하고 필수 환경변수가 설정되어 있는지 확인
- [ ] `pnpm install`로 의존성 설치 완료
- [ ] 개발 서버 실행: `pnpm dev`
- [ ] 브라우저에서 `http://localhost:3000` 접속 확인

## 📱 1. 기본 레이아웃 및 네비게이션

### 1.1 헤더/네비게이션 바
- [ ] **로고 표시**: "My Trip" 로고가 상단 왼쪽에 표시되는지 확인
- [ ] **검색창**: 데스크톱에서 헤더 중앙에 검색창이 보이는지 확인 (모바일에서는 숨김)
- [ ] **네비게이션 링크**: "홈", "통계", "북마크" 링크가 보이는지 확인
- [ ] **링크 클릭**: 각 링크를 클릭했을 때 해당 페이지로 이동하는지 확인
- [ ] **인증 버튼**: 로그인하지 않은 경우 "로그인" 버튼이 보이는지 확인
- [ ] **Sticky 헤더**: 스크롤해도 헤더가 상단에 고정되는지 확인

### 1.2 반응형 디자인
- [ ] **데스크톱**: 화면이 넓을 때 레이아웃이 적절히 표시되는지 확인
- [ ] **모바일**: 브라우저 개발자 도구로 모바일 뷰 확인 (검색창이 숨겨지는지 등)
- [ ] **태블릿**: 중간 크기 화면에서도 레이아웃이 깨지지 않는지 확인

## 🏠 2. 홈페이지 (`/`) - 관광지 목록

### 2.1 초기 로딩
- [ ] **페이지 접속**: 홈페이지(`/`)에 접속했을 때 로딩 상태가 표시되는지 확인
- [ ] **데이터 로드**: 관광지 목록이 로드되는지 확인 (기본값: 서울 지역, 전체 타입)
- [ ] **카드 표시**: 관광지 카드들이 그리드 형태로 표시되는지 확인

### 2.2 관광지 카드
- [ ] **썸네일 이미지**: 각 카드에 썸네일 이미지가 표시되는지 확인
- [ ] **이미지 없음 처리**: 이미지가 없는 관광지는 기본 아이콘이 표시되는지 확인
- [ ] **관광지명**: 카드에 관광지명이 표시되는지 확인
- [ ] **주소**: 주소 정보가 표시되는지 확인
- [ ] **타입 뱃지**: 관광 타입(관광지, 문화시설 등) 뱃지가 표시되는지 확인
- [ ] **반려동물 뱃지**: 반려동물 동반 가능한 경우 🐾 뱃지가 표시되는지 확인
- [ ] **호버 효과**: 카드에 마우스를 올렸을 때 확대/그림자 효과가 나타나는지 확인
- [ ] **카드 클릭**: 카드를 클릭했을 때 상세페이지로 이동하는지 확인

### 2.3 필터 기능
- [ ] **지역 필터**: 지역 선택 드롭다운이 작동하는지 확인
  - [ ] 서울, 부산, 경기 등 시/도 선택 가능한지 확인
  - [ ] "전체" 옵션 선택 시 모든 지역이 표시되는지 확인
  - [ ] 지역 변경 시 목록이 업데이트되는지 확인
- [ ] **타입 필터**: 관광 타입 선택 드롭다운이 작동하는지 확인
  - [ ] 관광지, 문화시설, 축제/행사 등 선택 가능한지 확인
  - [ ] "전체" 옵션 선택 시 모든 타입이 표시되는지 확인
  - [ ] 타입 변경 시 목록이 업데이트되는지 확인
- [ ] **반려동물 필터**: 반려동물 동반 가능 필터가 작동하는지 확인
  - [ ] 필터 활성화 시 반려동물 동반 가능한 관광지만 표시되는지 확인
- [ ] **필터 조합**: 지역 + 타입 + 반려동물 필터를 동시에 적용했을 때 정상 작동하는지 확인

### 2.4 검색 기능
- [ ] **검색창 입력**: 헤더의 검색창에 키워드를 입력할 수 있는지 확인
- [ ] **검색 실행**: Enter 키 또는 검색 버튼 클릭 시 검색이 실행되는지 확인
- [ ] **검색 결과**: 검색 결과가 목록에 표시되는지 확인
- [ ] **검색어 하이라이트**: 검색어가 결과에 강조 표시되는지 확인 (있는 경우)
- [ ] **검색 초기화**: 검색어를 지웠을 때 기본 목록으로 돌아가는지 확인
- [ ] **검색 + 필터**: 검색 결과에 필터를 적용했을 때 정상 작동하는지 확인

### 2.5 정렬 기능
- [ ] **정렬 옵션**: 정렬 드롭다운이 있는지 확인 (최신순, 이름순 등)
- [ ] **정렬 변경**: 정렬 옵션 변경 시 목록이 재정렬되는지 확인

### 2.6 무한 스크롤
- [ ] **스크롤 다운**: 페이지를 아래로 스크롤했을 때 추가 데이터가 로드되는지 확인
- [ ] **로딩 표시**: 추가 데이터 로딩 중 로딩 인디케이터가 표시되는지 확인
- [ ] **끝 도달**: 더 이상 데이터가 없을 때 적절한 메시지가 표시되는지 확인

### 2.7 지도 연동 (데스크톱)
- [ ] **지도 표시**: 데스크톱에서 지도가 화면 오른쪽에 표시되는지 확인
- [ ] **마커 표시**: 관광지 위치에 마커가 표시되는지 확인
- [ ] **마커 클릭**: 지도의 마커를 클릭했을 때 해당 관광지 카드가 강조되는지 확인
- [ ] **카드-지도 연동**: 리스트의 카드를 클릭했을 때 지도에서 해당 위치로 이동하는지 확인
- [ ] **인포윈도우**: 마커에 마우스를 올렸을 때 관광지명이 표시되는지 확인

### 2.8 지도 연동 (모바일)
- [ ] **탭 전환**: 모바일에서 "목록" / "지도" 탭이 있는지 확인
- [ ] **탭 클릭**: 탭을 클릭했을 때 뷰가 전환되는지 확인
- [ ] **지도 표시**: 지도 탭에서 지도가 전체 화면으로 표시되는지 확인
