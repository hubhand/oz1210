## 성능 확인

### 페이지 로딩 속도

- [ ] 홈페이지 로딩 시간이 3초 이내인가?
- [ ] 상세페이지 로딩 시간이 3초 이내인가?
- [ ] 이미지가 빠르게 로드되는가?

### Lighthouse 점수

**목표 지표**:

- Performance: > 80
- Accessibility: > 90
- Best Practices: > 90
- SEO: > 90

**확인 방법**:

1. Chrome DevTools → Lighthouse 탭
2. "Generate report" 클릭
3. 점수 확인

### Web Vitals

**목표 지표**:

- First Contentful Paint (FCP): < 1.8s
- Largest Contentful Paint (LCP): < 2.5s
- Time to Interactive (TTI): < 3.8s
- Cumulative Layout Shift (CLS): < 0.1

**확인 방법**:

1. Chrome DevTools → Performance 탭
2. 페이지 로드 후 "Record" 클릭
3. Web Vitals 측정

- [ ] 이미지가 lazy loading되는가?
- [ ] 이미지 크기가 적절한가?

## 문제 발생 시

테스트 중 문제가 발견되면:

1. **문제 기록**

   - 어떤 기능에서 문제가 발생했는가?
   - 에러 메시지는 무엇인가?
   - 재현 단계는 무엇인가?

2. **로그 확인**

   - Vercel Dashboard → Deployments → Function Logs
   - 브라우저 콘솔 (F12 → Console)

3. **해결 방법**
   - 로컬에서 동일한 문제 재현
